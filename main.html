How to Play

  1. Choose a game mode -- easy, medium or custom.
  2. The goal is to fill the 9x9 grid with a value of 1-9, with only one of each per block, column and row.
  3. To add a value to a cell, select it first, then use the buttons below or your keyboard. For keyboard users, "0" is clear.
  4. When in a bind, feel free to use the Live checker or Hints.

* For additional info, select Help. Happy Playing!

Game Play Options

Easy

These puzzles require the least amount of logic to solve, typically only simple scanning is needed. Use these when you really need a win or are short on time. Average solve time ranges from 5 to 20 minutes.

Medium

These puzzles require the use of more advanced techniques. Often, players need to use pencil marks as well as counting rows and columns. Hints will become your best friend if you get stuck. These puzzles can take over 20 minutes.

Custom

See a puzzle in a newspaper, online or somewhere else? Feel free to enter the values and play here or check the answers by selecting the Solve option.

Hints

These clues are generated by scanning each block, column and row that a cell belongs to. By elimating the values that are used, the hints show the possible values for that cell. Note that these can be your best friend when correct values are entered but can become your worst enemy when incorrect values are entered.

Live Checker

Before a puzzle begins, some logic is appied under the hood to ensure that a solution is found. These expected values are stored and compared when checking the status. Any incorrect values are hightlighted in red. Fix these values to get back on track.

Solve

Even though solving on your own is the most rewarding, somtimes we all just want the answers. This option is for anyone looking for a way out!
